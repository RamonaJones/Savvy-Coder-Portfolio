My Coding Camp Journey

The Mind of A Frontend Developer

Learning something for the first time can be nerve wrecking, especially if it's coding. Even if you have a level of familiarity, relearning a skill can be nerving. This blog shares my coding camp journey through Savvy Coders. One thing that I know is that you have to give 100% and more and engage your whole self into learning. It also takes consistent, correct practice to come to master. I'm ready to learn so I can grow and earn!
